## What is Vibma?

Vibma connects AI agents (like Claude) to Figma through the Model Context Protocol (MCP). It provides **81 tools** across 12 domains — from creating shapes and frames to managing design tokens, variables, and component variants.

## Architecture

```
Claude Code ←stdio→ MCP Server ←WebSocket→ Relay ←WebSocket→ Figma Plugin
```

The MCP server exposes tools that AI agents call. Commands flow through a WebSocket relay to the Figma plugin, which executes them against the Figma API and returns results.
